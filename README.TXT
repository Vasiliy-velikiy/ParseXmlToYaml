ЗАДАНИЕ:Необходимо написать консольное Java приложение, считывающее XML файл sample.xml  и создающее структуру
 объектов на его основе, после чего выполняющее выгрузку структуры объектов в YAML файл output.yml.
        При выгрузке необходимо удалить элемент Product.TechRegs. В результате запуска программы должен
        сформироваться файл output.yml, содержащий данные, идентичные данным файла sample.xml, но в формате YAML
        и без элемента Product.TechRegs. На способ реализации задачи ограничений нет.




ОПИСАНИЕ ПРОЕКТА: Данная программа использует библиотеки underscore-java и snakeyaml  для конвертации из xml в yml
Тестовый файл sample.xml лежит в корне проекта, там же будет создаваться файл output.yml после завершения программы с
 вырезанной секцией <TechRegs>.
и его дочерними элементами( в задании не указано про дечерние эл-ты но я решил их вырезать). Если программа запущена
 повторно, она просто перезапишет файл output.yml
 Я оставил output.yml как результат отработки программы

 ТРЕБОВАНИЯ: JAVA 1.8
 в мавене подлюченная библиотека         <dependency>
                                 			<groupId>com.github.javadev</groupId>
                                 			<artifactId>underscore</artifactId>
                                 			<version>1.78</version>
                                 		</dependency>
